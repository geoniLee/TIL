코딩 테스트 완전 정복<br>
* 변수: 데이터를 저장하고 처리하는데 사용되는 구성요소
    - 어떤 종류의 데이터를 저장할 것인지 결정해야 합니다
    - 변수에 사용할 이름을 정해야 합니다(변수의 이름은 유일해야 합니다.)
* 스칼라: 하나의 데이터를 저장하는 타입
    - 정수형: 소수점 없는 숫자(int)
    - 부동 소수점: 소수점을 포함하는 숫자를 저장(flaot, double)
    - 논리형: 참 또는 거짓 중 하나의 값을 저장
    - 문자형: ASCII코드표에 정의된 하나의 문자를 저장하는데 사용
* 배열: 하나의 이름으로 동일한 타입의 변수를 묶어서 관리
    - 임의 접근이 가능함
    - 스칼라 타입처럼 대입할 수 없음
* 문자열: 문자들이 연속적으로 나열된 데이터
    - 문자열 찾기: find(찾으려는 문자열 혹은 문자) 
    - 문자열 추가: '+', '+='을 사용 가능
    - 문자열 수정: '[]', replace() 사용 가능
    <br>
    - 자주쓰는 문자열<br>
    length()<br>
    size()<br>
    empty()<br>
    insert(pos, str)<br>
    erase(pos, len)<br>replace(pos, len, str)<br>
    substr(pos, len)<br>
    compare(str)<br>
    push/pop()<br>
    resize(n)<br>
    swap(교환할 문자열)<br>
    toupper/tolower(변환할 문자)
    
* 자주하는 실수
    * 단락평가(and, or)
    * off by one: 의도한 횟수보다 한 번 더 많거나 적게 반복됨
    * 0으로 나누기
    * 오버 플로(변수의 범위)

* 숫자 다루기(cmath.h)
    * 반올림 하기:round(값)
    * 올림하기: cell(값)
    * 소수점 버리기/ 내림: flloe(값)

* 문자열 스트림
    * stringstream: 문자열을 마치 스트림처럼 다룰 수 있도록 해주는 클래스
    * 공백 기준으로 분리<br>
    1. 분리할 문자열을 stringstream의 생성자에 전달하여 입력 스트림으로 설정
    2. \>\> 연산자를 이용해 문자열에서 원하는 데이터를 차례대로 변수에 저장
    * 특정 문자 기준으로 분리<br>
    (getline(입력 스트림, 토큰을 저장할 문자열 변수, 구분자 문자))
     ```Cpp
    string text = "apple,banana,grape,orange";
    stringstream ss(text);
    string token;

    while (getline(ss, token, ',')) {
        cout << token << endl;
    }
     ```
