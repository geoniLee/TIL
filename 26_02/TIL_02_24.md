    **빅오 표기법**: 최악의 경우를 기준으로 점근적 표기법을 활용해서 성능을 표현하는 방법

    **최소한의 상한 구하기**<br>
    - 가장 많은 영향을 미치는 부분만 남기고 앞에 있는 상수를 없애기

    **자주 보는 시간 복잡도**<br>
    1. 1차원 배열 -> O(N)
    2. 2차원 배열 -> O(N^2)
    3. 이진 탐색 -> O(log N)
    4. N개의 원소로 만들 수 있는 모든 조합 -> O(2^N)

* C/C++
    * 변수: 변할 수 있는 수
    * unsigned vs signed (부호 없음 vs 부호 있음)
    * 오버 플로우: 자료형이 표현 가능한 수를 넘어서는 경우
<br><br>
    * 인코딩: 형태 A에서 B로 변환하는 것
    * 디코딩: 형태 B에서 A로 변환하는 것
    * ASCII: 문자 형태의 데이터와 숫자 형태의 데이터 사이의 인코딩 규약 중 하나
<br><br>
    * scanf(): 키보드로부터 데이터를 입력 받을 수 있게끔 해주는 함수
<br><br>
    * 연산자 우선순위와 결합법칙 > 연산자들 간에는 우선순위가 존재
    * 정수 피연산자와 실수 피연산자 > int 자료형 끼리의 나눗셈은 결과도 int, float 자료형 끼리의 나눗셈은 결과도 float
<br><br>
    * 복합 대입 연산자: 산술 연산과 대입 연산이 함께 계산되는 연산자
    * 증감 연산자: 단항의 피연산자를 가짐(ex. ++, --)
    * 전치 -> 지금 당장 연산, 후치 -> 다음 줄에 연산
    * 논리 연산자: 불대수(boolean) 연산자 (&&, ||, !)
    * 관계 연산자: 피연산자 간의 관계를 판단하는 연산자
<br><br>
    * 형변환 연산자: 자료형 A에서 B로 변환시켜주는 연산자
    * 삼항 연산자: 피연산자로 3개의 항을 갖는 연산자<br>
    조건식 ? 참일 경우 : 거짓일 경우
    * sizeof: 피연산자의 자료형 크기를 반환함(ex. sizeof(피연산자))
<br><br>
    * 비트 연산자: 비트별로 논리 연산을 수행하는 연산자<br>
    논리 곱 연산자(&)<br>
    논리 합 연산자(|)<br>
    논리 반전 연산자(~): 전체 비트를 반전<br>
    논리 베타합 연산자(^): 비트별로 1의 개수가 홀수개면 1, 짝수개면 0<br>
    * 비트 이동 연산자: 피연산자를 2진수로 모두 평가한 후 2진수 기준 왼쪽 오른쪽으로 n칸씩 이동시키는 연산자(<<, >>)<br>
    왼쪽 -> 곱하기 2, 오른쪽 -> 나누기 2
