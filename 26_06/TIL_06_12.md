<h3>언리얼 심화 라이브세션</h3>
<h4>GAS
</h4>

* PlugIn - Gameplay Abilities 활성화

* GAS
    * Tag -> Bool
    * 변수 저장(HP, MP) - > Attributr Set
    * GameplayEffect
    * 사운드 이펙트 -> GameplayCue

* GameAbility
    * Tag
        * Assert Tag
            
            해당 GA의 태그
        * 태그 포함 어빌리티 취소   

            해당 GA 실행동안 설정한 태그를 가진 어빌리티를 강제 종료
        * 태그 있는 능력 차단

            해당 GA실행동안 설정한 태그를 가진 GA가 실행되지 않도록 함
        * 활성화 소유 태그

            해당 GA가 실행되는 동안 부착되는 Tag
        * 활성화 필수 태그

            해당 태그를 가지고 있어야 해당 GA 실행 가능
        * 소스 관련 2종

            GA를 주는 주체가 가진 Tag에 따라 동작
        * 타켓 관련 2종
            상대방이 가진 Tag에 따라 동작

    * Instance 정책
        * None Instance: 함수 처럼 실행(원본을 돌려씀)
        * Instanced Per Actor: 사용하는 Actor마다 Instance 생성
        * Instanced Per Execution: 실행할 때마다 Instance 생성
