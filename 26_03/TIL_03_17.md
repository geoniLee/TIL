<h3>CH02 개인 과제 5번</h3>
<h4>Unreal Engine Cpp 활용 프로그램 제작</h4>

* Unreal Engine API
    * GEngine: 엔진 기능에 접근할 때 쓰는 전역 엔진 객체
    * FString: 언리얼에서 쓰는 문자열 타입(조작 가능한 문자열 클래스)
    * FColor: 언리얼의 색상 타입(채널당 8비트 정밀도의 색상을 저장)
    * FMath: 언리얼에서 제공하는 수학/계산 관련 기능 모음

* AddOnScreenDebugMessage<br>
-> 화면에 메시지 출력
    * GEngine: UEngine 타입의 전역 포인터
        ```
        #include "Engine/Engine.h"선언 필수

        //사용 예
        GEngine->AddOnScreenDebugMessage(메시지 식별자, 출력할 시간, 색상, 출력할 내용, bNewerOnTop, TextScale)
        ```
        * 메시지 식별자
            * -1을 주면 항상 새 메시지 추가
            * 같은 키를 주면 기존 메시지를 덮어 씌움
        * 색상
            * "FColor::색상" 으로 사용
        * 출력할 내용
            ```
            ex. FString::Printf(TEXT("Actor : %s"), GetName());
            ```
        * bNewerOnTop(선택)
            * true면 새 메시지가 위에 표시 
            * 기본값은 true
        * TextScale
            * 글자 크기 배율
            * ex. FVector2D(1.5f, 1.5f)


* 구현 사항
    * Move()
        * x,y,z에 0~10 범위의 랜덤값 연산
        * 이전 좌표와 비교해 거리를 구하고 총 이동거리에 합산
        * 이동 후 위치와 이번 이동 거리 출력
        * 일정 확률(50%)로 이벤트 호출
    * Turn()
        * x,y,z에 -10~10 범위의 랜덤값 연산
        * 회전 후 상태를 출력
    * TriggerEvent()
    * BeginPlay -> Tick
        * 각 이동 사이에 지연을 주기 위해 Tick으로 이동
        * deltatIime을 이용해 시간 계산 후 사용
        * 마지막 반복 때 총 이동 거리와 이벤트 발동 횟수 출력
