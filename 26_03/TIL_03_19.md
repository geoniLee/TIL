<h3>Cpp 코딩 테스트 완전 정복</h3>
<h4>정렬</h4>

* 정렬하는 이유
    1. 원하는 값을 빨리 찾을 수 있음
    2. 정렬이 전제 조건인 알고리즘이 존재함
        * 이진 탐색
        * upper_bound / lower_bound
        * ...

* 정렬
    * 삽입 정렬<br>
    정렬되지 않은 데이터를 이미 정렬된 부분 배열의 적절한 위치에 삽입하여 데이터를 정렬하는 알고리즘
        * 시간 복잡도 - O(N^2)
        * 세부 동작
            1. key을 왼쪽과 비교하며 키가 왼쪽 값보다 작을 경우 왼쪽으로 이동
            2. 더 이상 비교할 값이 없거나 key 가 왼쪽 값보다 크면 다음 key로 이동
    
    * 병합 정렬<br>
    정렬되지 않은 데이터를 반으로 나눈 뒤, 작은 단위부터 정렬하며 병합해 나가는 정렬 알고리즘
        * 세부 동작
            1. 분할: 중간 지점을 기준으로 2개의 하위 배열로 나눕니다.
            2. 정복: 재귀적으로 병합 정렬을 호출해서 정렬합니다
            3. 결합: 정렬된 2개의 하위 배열을 순서대로 비교하여 작은 값을 먼저 새로운 배열에 추가합니다.
        * 시간 복잡도 - O(NlogN)

    * 계수 정렬<br>
    각 값의 빈도수를 세어 그 정보를 기반으로 정렬을 수행
        * 세부 동작
            1. 카운트 배열 생성: 배열의 최대값 k를 기준으로 K+1 인 배열 생성
            2. 빈도 수 계산: 입력 배열을 순회하며 각 원소의 값을 인덱스로 사용하여 빈도수를 누적
        * 시간 복잡도 - O(N+K)
        * 제한 사항
            * 음수 값이 존재하는 경우 
            * 원소 값의 범위가 너무 넓거나 듬성듬성 있는 경우

    * 힙 정렬<br>
    힙을 사용한 정렬 방법
        * 힙: 조건을 만족하는 이진 트리(최대 힙, 최소 힙)
            * max_heapify(val): 특정 원소를 기준으로 최대 힙을 만족하도록 재구성하는 함수
        * 힙 정렬: 최대 값을 하나씩 꺼내서 배열의 뒤쪽에 쌓아가는 정렬 알고리즘
        * 시간 복잡도 - O(logN)
