<h4>백트래킹</h4>

* 백트래킹<br>
완전 탐색의 한계를 극복하는 기법 / 유망 함수를 활용하여 불필요한 탐색을 가지치기하는 과정
    * 제약 만족 문제 - 주어진 제약 조건을 만족하는 해답을 찾는 문제

    * 유망 함수: 제약 조건 검사기 - 현재까지의 부분 해답이 문제의 제약 조건을 만족할 가능성이 있는지를 검사합니다.
        1. 해의 후보 정의: 문제를 풀기 위한 해가 될 수 있는 잠재적 후보를 정의함
        2. 상태 - 공간 트리로 표현
        3. 유망함수 정의
    
    * 부분 집합 합 - 1~N 까지 숫자를 조합하여, 그 합이 정확히 K가 되는 조합을 찾는 문제
        * 완전 탐색 - 모든 경우의 수 탐색(O(2^N))
        * 백트래킹 - 불필요한 경로를 조기에 차단하여 탐색 시간을 줄일 수 있음
    * N-퀸 문제 -  N x N 체스판에 N개의 퀸을 배치 했을 때 서로를 공격할 수 없는 위치에 놓을 수 있는 방법이 있는지 찾는 문제
        * 완전 탐색 - 모든 경우의 수 탐색 (O(N^N))
        * 백트래킹 - 퀸이 추가될 때마다 행/대각선 방향에 겹치는 여왕이 있으면 더 탐색하지 않음

<h4>최단 경로 알고리즘</h4>

* 최단 경로 알고리즘<br>
그래프 내의 두 노드 사이를 이동할 때, 간선들의 가중치 총합이 최소가 되도록 하는 알고리즘

* 다익스드라 알고리즘<br>
가중치가 있는 그래프에서 하나의 노드로부터 다른 모든 노드까지의 최단 경로를 찾는 알고리즘
    1. 최적 부분 구조 - 찾을 최단 경로는, 그 일부를 이루는 최단 경로들로 구성된다는 원리
    2. 그리디 속성

    * 세부 동작
        1. 최단 거리 배열 = 시작 노드로부터 각 노드까지의 최단 경로 비용
        2. 직전 노드 배열 = 각 노드에 도달하기 위한 최단 경로 상에서의 바로 이전 노드

    * 한계 
        1. 모든 간선이 음의 가중치가 아닐때 제대로 동작함

* 벨만 포드 알고리즘<br>
매 단계 마다 모든 간선의 가중치를 다시 확인하여 최소 비용을 갱신 - 음의 가중치가 있어도 최단 경로 탐색 가능
