<h3>게임 개발자를 위한 Cpp 문법</h3>

* iostream - Cpp 표준 라이브러리의 입출력 기능을 제공하는 헤더 파일

* 변수(int, float/double, char, bool, string, unsigned)

* 연산자
    1. 일반 연산(+, -, *, /, %)
    2. 비교 연산자(==, !=, >, <, >=, <=)
    3. 대입 연산자(=, +=, -=, *=, /=, %=)
    4. 전위 연산(++a), 후위 연산(a++)

* 배열: 동일한 타입의 여러 값을 한번에 담을 수 있는 자료구조

* 함수: 반복되는 코드나 논리를 이름을 붙여 정리하고 가져다 쓸수 있도록 함<br>
구성 요소
    
    1. 인자
    2. 동작
    3. 반환
    4. 이름

* 조건문
    
    1. if
    2. if-else
    3. if-else if-else

    * 복합조건 - (&&:and, ||:or)

    4. for문: 반복의 범위가 명확하게 주어질 때 사용
    5. while문: 특정 조건을 확인하며 반복하는 경우
