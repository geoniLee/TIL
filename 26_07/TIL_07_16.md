# TIL 7.16
<h4>ch04 팀프로젝트</h4>

1. 아이템 사운드 출력 구조 추가
    * RandomItemGrantActor에서 아이템 획득 성공 시 Pickup 이펙트와 사운드를 같이 처리할 수 있도록 정리.
    * Obstacle은 실제 생성 성공 시점에 Spawn Sound가 나도록 ObstacleBase 쪽에서 처리하는 구조로 잡음.
2. 투사체 사운드 세분화
    * Spanner는 “플레이어에게 Hit 되었을 때” 사운드가 나야 하므로 HitTargetSound 성격으로 분리.
    * Bomb은 “제거/폭발 시점”에 사운드가 나야 하므로 FinishedSound 성격으로 분리.
    * Projectile 비행 중 루프 사운드를 Cosmetic Projectile에 붙여 재생하고, 종료 시 정리하는 흐름도 추가됨.
3. Bomb 시각 효과 보강
    * Bomb Material에 Fresnel/Charge 계열 표현을 넣고, ExplosionChargeAmount 값을 시간에 따라 증가시키는 방식으로 폭발 전 시각적 변화를 줄 수 있게 함.
    * Preview에는 적용하지 않고 Cosmetic에서만 동작하도록 방향을 잡음.
    * Bomb 종료 시점에는 기존 Cascade/NS 양쪽을 대응할 수 있도록 준비함.
4. Buff / Debuff GameplayCue 사운드 세팅
    * Buff/Debuff GameplayCueNotify 쪽에 1회성 시작 사운드를 넣는 방향으로 정리.
    * On Burst 시점에서 Spawn Sound Attached를 사용해 플레이어에게 붙어서 재생되도록 설정하는 흐름을 확인함.
    * 시간 갱신마다 반복 재생되는 구조가 아니라, 적용 시작 시 1회만 재생되는 구조로 잡음.
