<h3>unordered_set과 pair< , ></h3>

* unordered_set이 키를 저장하는 방식
    
    -> unordered_set<Key, Hash, KeyEqual>
    
    * Hash: 값을 size_t 정수로 변환
    * KeyEqual: 두 값이 실제로 같은지 비교

    ---
    * int등 산술형, 열거형, 포인터 형 등에는 표준 해시 제공
    * pair< , >에서 `hash<pair<int, int>>`는 표준에서 기본 제공되지 않음 

    ---
    * 직접 해시 함수를 제공하는 방법
    ```
    struct PairHash {
        size_t operator()(const pair<int, int>& p) const noexcept {
            size_t h1 = hash<int>{}(p.first);
            size_t h2 = hash<int>{}(p.second);

            return h1 ^ (h2 << 1);
        }
    };
    ```

    이후 `unordered_set<pair<int, int>, PairHash> cnt;`로 사용
