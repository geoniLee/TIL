# TIL 7.03
<h4>Ch04 팀 프로젝트</h4>

## 아이템 사용 입력 서버 연동

- 아이템 사용 입력을 서버 흐름과 연결
- 아이템 사용 시 보유 중인 아이템 Ability가 실행되도록 처리
- 클라이언트 입력 이후 서버에서 아이템 슬롯 소모가 진행되도록 RPC 흐름 정리

## 아이템 슬롯 서버 동기화

- 아이템 슬롯 정보를 서버 기준으로 관리하도록 수정
- 아이템 획득 시 서버에서 슬롯에 아이템 추가
- 아이템 사용 후 서버에서 슬롯 제거 처리
- 슬롯 변경 내용이 소유 클라이언트에 복제되도록 처리

## 아이템 Gameplay Ability 구조 정리

- 아이템 Ability 공통 베이스 클래스 `GA_SGItemBase` 추가
- 입력 Release 대기 로직을 공통 베이스로 분리
- 로컬 입력 처리와 서버 실행 흐름을 공통화
- 각 아이템 Ability는 실제 효과 실행 로직 중심으로 정리

## 아이템 Ability 수정

- 공통 베이스 흐름에 맞게 각 아이템 Ability 실행 구조 변경

## 아이템 Effect / Blueprint 설정 수정

- SpeedUp / SpeedDown 관련 GameplayEffect 설정 수정
- RandomItemGrantActor Blueprint 설정 수정
- 아이템 사용 테스트에 필요한 에셋 설정 반영

## 아이템 소모 시 ASC Ability 정리

- 아이템 소모 후에도 ASC에 AbilitySpec이 남는 문제 확인
- 아이템 소모 시 저장된 `AbilityHandle`을 이용해 ASC에서 Ability 제거 처리
- Ability가 활성 중이면 `SetRemoveAbilityOnEnd()`로 종료 시 제거 예약
- 비활성 상태면 `ClearAbility()`로 즉시 제거
