<h4>Generate Binary Strings Without Adjacent Zeros</h4>

* 알고리즘
    * 백트래킹
    * DFS

* 아이디어
    1. string에 재귀함수로 0 -> 1을 push_back, pop_back을 하여 string을 완성
    2. string의 길이가 n인 경우 res에 push_back 한 이후 return
    3. string에서 마지막 값이 0인 경우 이번에 0을 push/pop 작업을 스킵
    4. 완성된 res을 반환

<h4>CH04 팀프로젝트</h4>

* 투사체 Listen Server 대응 시각화
    * 문제 상황
        1. Server에서 투사체를 날리고 Client에 복제
        2. 복제 간격 + 빠른 투사체의 속도 + 한 PC에서 여러개의 Client을 띄워 확인
        3. 2번의 환경과 맞물려 Host가 아닌 곳에서 끊김 현상이 발생
    
    * 해결방안
        * 서버 판정용 projectile과 로컬 시각용 Preojectile을 분리
            * 서버 판정용 
                * 보이지 않음
                * 판정을 로컬에 전달
                * 클라이언트에는 발사위치와 속도만 복제
                * 충돌 감지 및 처리를 함
            * 로컬 시각용 
                * 다른 플레이어/ 서버에 복제되지 않음
                * 복제된 발사위치와 속도를 사용해 발사
                * 충돌감지 및 처리를 하지 않음
