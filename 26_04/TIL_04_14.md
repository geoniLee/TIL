<h3>Cpp과 Unreal Engine으로 3D 게임 개발 </h3>
<h4>Character 클래스를 활용한 캐릭터 구현하기</h4>

* GameMode
    * GameMode: 총괄 관리자
        1. 플레이어 캐릭터(Pawn/Character) 관리
        2. PlayerController 클래스
        3. 게임 규칙 관리 - 로직
        4. 클래스
            1. GameState 클래스 - 게임 전역 데이터
            2. PlayerState 클래스 - 개별 캐릭터마다의 데이터
    * GameModeBase: 멀티 플레이 로직 제공x / 싱글 플레이 및 직접 멀티 플레이 로직을 구현할 때 사용
    * GameMode: 멀티 플레이제공 /  GameState, PlayerState 연결이 활성화

* Pawn
    * Pawn: 플레이어/ Actor가 소유할 수 있는 가장 상위 클래스
        * 주로 기존 캐릭터의 이동 방식을 벗어난 특수한 로직을 자유롭게 구현할 때 사용
    * Character: 
        * Pawn을 상속받아 만들어진 자식 클래스
            * 이동, 회전, 점프, 중력, 지형 따라가기, 네트워크 동기화 등의 기능이 구현되어 있음
            * 캐릭터를 구성하는 전형적인 요소들이 표준화 되어 있음
        ---
        * CapsuleComponent (Root Component): Collision 컴포넌트(캡슐)
        * ArrowComponent: 캐릭터가 보는 방향을 알려줌
        * SkeletalMeshComponent: 캐릭터의 3D 모델과 애니메이션을 적용하는 컴포넌트
            * SkeletalMesh Mesh에 모델링 지정
        * CharacterMovementComponent: 캐릭터의 이동, 점프,  중력, 네트워크 동기화 등의 물리적 이동 로직을 담당하는 핵심 컴포넌트

* Camera 및 GameMode 설정
    * SpringArmComponent
        1. USpringArmComponent* 생성
            * include "Camera/CameraComponent.h"
        2. RootComponent에 부착
        3. 캐릭터와 카메라 사이의 거리
            >SpringArmComp->TargetArmLength()
        4. 컨트롤러 회전에 따라 카메라도 회전하도록 설정
            >SpringArmComp->bUsePawnControlRotation = true; 
    * CameraComponent
        1. CameraComponent* 생성
        2.  SpringArm에 부착
            * 스프링암의 소켓 위치에 카메라 부착
            >USpringArmComponent::SocketName
        * 컨트롤러 회전에 따라 카메라도 회전하지 않도록 설정
            >SpringArmComp->bUsePawnControlRotation = false;
