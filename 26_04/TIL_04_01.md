<h3>C++ 코딩테스트 완전 정복</h3>
<h4>그리디 알고리즘</h4>

* 그리디 알고리즘
    <br>
    각 단계에서 지역적으로 최적의 선택을 하여 전체적으로 최적해를 찾는 것을 목표로 함

    * 최적 부분 구조
    <br>
    전체 문제의 최적해가 그 하위 문제들의 최적해로 구성되는 성질

    * 그리디 선택 속성
    <br> 각 단계에서의 자연적인 최적 선택이 전체 문제의 최적해로 이어져야 한다는 성질

    * 최소 신장 트리
    <br>
    주어진 그래프에서 모든 정점을 포함하면서, 사이클 없이 하나의 연결된 구조를 이루는 부분 그래프
        - 이 때 간선의 개수는 항상 정점의 개수보다 하나 적음
    * 최소 신장 트리 - 프림 알고리즘
        <br>
        과정
        1. 임의의 정점을 하나 선택하여 최소 신장 트리에 추가
        2. 현재 트리에서 연결 가능한 간선 중, 가중치가 가장 작은 간선을 선택하여 트리에 추가
        3. 위 과정을 모든 정점이 트리에 포함될 때까지 반복

    * 최소 신장 트리 - 크루스칼 알고리즘
        <br>
        과정
        1. 그래프의 모든 간선을 가중치 기준으로 오름 차순으로 정렬
        2. 가중치가 가장 작은 간선부터 하나씩 선택하여, 사이클이 발생하지 않으면 최소 신장 트리에 추가
        3. 위의 과정을 모든 정점이 연결될 때까지 반복
