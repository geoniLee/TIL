<h4>합승 택시 요금</h4>

* 알고리즘
    * 다익스트라
    * 우선순위 큐
    * 최단 경로

* 아이디어
    1. 출력 값 = (시작 지점부터 특정 노트 까지의 최솟값 + 특정노드에서 A/B 까지 최솟값의 합)중 최솟값
    2. 다익스트라로 시작 지점 -> 각 노드까지의 최소 비용을 구해 저장
    3. a->b 까지 가는 비용과 b->a까지 가는 비용이 같음
        1. a -> 각 노드까지의 최소 비용을 구해 저장
        2. b -> 각 노드까지의 최소 비용을 구해 저장
    4. 1부터 시작해서 모든 노드를 순회하며 s -> 해당 노드 + 해당노드 -> a + 해당노드 -> b의 최솟값을 갱신
