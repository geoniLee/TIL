<h3>Cpp과 Unreal Engine으로 3D 게임 개발 </h3>
<h4>Enhanced Input System을 활용한 입력 매핑 구현하기</h4>

* PlayerController
    1. 입력 처리 - EnhancedInputSystem
    2. 카메라 제어 로직
    3. UI와의 상호작용
    4. Possess(빙의)

* EnhancedInputSystem
    * InputMappingSystem(IMC)
        > IA들을 총괄해서 관리
    * InputAction(IA)
        > 추상적인 행동
        * ValueType(값 타입)
            * digital bool => on / off 
            * Axis1D(float) => 전진 / 후진
            * Axis2D(Vector2D) => x,y축 처리
            * Axis3D(Vector) => x,y,z축 처리
        * Triggers(트리거)
            > 입력이 활성화 되는 조건
            * Press Trigger: 키를 누르는 순간에 작동
            * Hold Trigger: 키를 일정 시간 눌렀을 때 작동
            * Released trigger: 키를 뗄 때 작동
        * Modifier(모디파이어)
            > 입력 값을 수정하거나 변환하기 위한 설정
            * Negate(부정): 반전
            * Scaler(스칼라): 입력값에 일정 배율을 곱해줌
            * Swizzle Input Axis Value(스위즐 입력 축 값): 입력값의 축을 다른 축으로 쓸 때 사용
        
* PlayerController
    * 헤더
        * UInputMappingContext* 선언 (전방 선언 필요)
        * 사용할 IA선언 (전방 선언 필요)
            > ex. UInputAction* MoveAction;
    * cpp
        1. PlayerController에 연결된 LocalPlayer 객체를 가져옴
            > if (ULocalPlayer* LocalPlayer = GetLocalPlayer())
        2. LocalPlayer에서 EnhancedLocalPlayerSubSystem 획득
            > if (UEnhancedInputLocalPlayerSubsystem* Subsystem = LocalPlayer->GetSubsystem<UEnhancedInputLocalPlayerSubsystem>())
        3. SubSystem을 통해 할당한 IMC을 활성화 
            > Subsystem->AddMappingContext(InputMappingContext, 0);
            * 0은 우선순위
